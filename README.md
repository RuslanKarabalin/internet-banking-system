# go-bank-system

## Описание

Простой пет-проект для тренировки навыков разработка на golang.

Коротко о самом проекте:
небольшая банковская система,
есть пользователи,
у пользователей есть счета,
у каждого счёта может быть несколько карт,
пользователи могут делать переводы между счетами.
